# Todo App

This is a simple Todo application built with Java and Spring Boot.
The project was created to test Tool Calling with ChatGPT for managing todos via chat.

**N.B.** Before running the application, you should add your own **OPENAI_API_KEY** as an environment variable to connect to your OpenAI API.

## Technology Stack

*   **Language:** Java 21
*   **Framework:** Spring Boot
*   **Database:** H2
*   **AI LLM:**  chatgpt
*   **Build Tool:** Maven
